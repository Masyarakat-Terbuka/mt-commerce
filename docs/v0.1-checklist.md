# v0.1 Foundation

This document outlines the work for the first release of mt-commerce, **v0.1 Foundation**.

It is a working list of the pieces that need to come together for a small business to run a real store on the platform: products, checkout, payments, shipping, and notifications, all in Bahasa Indonesia, deployable on a modest server.

Each item is sized to be a self-contained piece of work, suitable for a single GitHub issue. The list is grouped into streams that can largely progress in parallel. Items within a stream are roughly ordered by dependency.

This document will be retired once the work is tracked in GitHub issues and projects. Until then, it serves as the shared map.

## Notes on this list

The list is ambitious. Some streams may grow, some may shrink, and some items may move to a later release as the work meets reality. The shape of the foundation is what matters; the order and scope of the items are not fixed.

If the list feels too large, that is honest information about the size of building a commerce platform from scratch. We err on the side of writing things down so the work is visible, then deciding together what fits in the first release and what waits.
